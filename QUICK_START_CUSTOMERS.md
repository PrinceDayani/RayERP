## üîç Troubleshooting

### Problem: "No customers found" in dropdown

**Solution A**: Create customer from invoice page
```
1. Click "+ Add" button in invoice page
2. Fill in customer details
3. Customer is automatically marked as customer
```

**Solution B**: Check existing contacts
```
1. Go to Contacts page
2. Edit each contact you want as customer
3. Check "Customer Status" checkbox
4. Save
```

**Solution C**: Verify contact status
```
1. Go to Contacts page
2. Look for "Customer Status" indicator on contact cards
3. If not checked, edit and check it
```

### Problem: Customer not appearing after creation

**Check**:
- ‚úì Is "Customer Status" checkbox checked?
- ‚úì Is contact status "Active"?
- ‚úì Did you save the contact?
- ‚úì Try refreshing the invoice page

**Fix**:
```
1. Go back to Contacts
2. Find the contact
3. Edit it
4. Verify "Customer Status" is checked
5. Verify "Status" is "Active"
6. Save again
7. Refresh invoice page
```

### Problem: Created customer but can't find it

**Solution**:
```
1. Go to Contacts page
2. Use search bar to find customer by name
3. Click on customer card
4. Verify these fields:
   - Customer Status: ‚úì (checked)
   - Status: Active
   - Visibility Level: Universal or Personal
5. If any is wrong, edit and fix
```

## üí° Pro Tips

### Tip 1: Use Contact Types
When creating customers, set "Contact Type" to:
- **Client**: For regular customers
- **Company**: For business customers
- **Partner**: For business partners

### Tip 2: Set Priority
Mark important customers:
- **High**: VIP customers
- **Critical**: Key accounts
- **Medium**: Regular customers
- **Low**: Occasional customers

### Tip 3: Add Tags
Use tags for easy filtering:
- "VIP"
- "Regular"
- "Wholesale"
- "Retail"
- "International"

### Tip 4: Use Visibility Levels
- **Universal**: Everyone in company can see
- **Departmental**: Only your department can see
- **Personal**: Only you can see

### Tip 5: Quick Customer Creation
From invoice page:
1. Click "+ Add"
2. Fill minimum fields (name, phone)
3. Click "Add Customer"
4. Edit later to add more details
